<!--more-->

后缀名 意味着什么文件

Extension	| Meaning 
-----		| ------------- 
.c 			| C language source file  
.cc,.cpp 	| c++ language source file
.h 			| Header file
.m 			| objective-c source file
.mm			| objective-c++ source file
.pl			| Perl source file（奇怪了，objc里面也用perl？）
.o 			| object(compiled)file
